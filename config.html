使用说明

如何修改配置？

编辑项目中的 config.js 文件，修改以下配置项：

GitHub配置

github: {
    owner: '你的用户名',
    repo: '仓库名',
    contentPaths: {
        'post': 'content/post',
        'page': 'content/page'
    },
    dataPath: 'data'
}

API代理配置

api: {
    proxyUrl: 'https://gitmd-api.xxx.workers.dev',
    useProxy: true
}

获取GitHub Token

  1. 访问 GitHub Token设置
  2. 点击 "Generate new token (classic)"
  3. 勾选 repo 权限
  4. 生成并复制Token
  5. 在Cloudflare Worker环境变量中设置 GITHUB_TOKEN

部署Cloudflare Worker

  1. 访问 Cloudflare Dashboard
  2. Workers & Pages → Create Worker
  3. 复制 worker/worker.js 代码
  4. 设置环境变量 GITHUB_TOKEN
  5. 获取Worker URL并填入config.js
